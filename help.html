Help & About

Are my passwords kept safe?

All passwords are hashed and the hash is stored, then the password is deleted

I forgot my password. Now what?

All passwords are hashed and the hash is stored, then the password is deleted. There is no way to recover your password if you forget it. Please email me. I can move your files to a temporary location, delete your username, and then you can create your username again, and then I can move your files back.

Can I give two files the same name?

If you upload a file with the same name as an existing one, it will overwrite the existing one. Comments and marks will also be lost.
